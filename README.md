# 📊 Dashboard de Performance Comercial Global

## 📌 Visão Geral do Projeto
Este projeto foi desenvolvido como parte dos meus estudos em *Data Analytics*. O objetivo principal foi transformar dados brutos de vendas mundiais em um painel estratégico, permitindo a análise de KPIs fundamentais para a tomada de decisão comercial.

O dashboard cobre o período de 2011 a 2014, apresentando métricas de vendas, lucro e comportamento de descontos por região e categoria.

## 🚀 Tecnologias Utilizadas
* *Power BI*: Criação de dashboards e visualização de dados.
* *DAX (Data Analysis Expressions)*: Criação de medidas calculadas para análise de lucro e margem.
* *Power Query*: Processo de ETL (Extração, Transformação e Carga) para limpeza e tratamento dos dados.
* *Git/GitHub*: Controle de versionamento e portfólio.

## 📈 Principais Indicadores (KPIs)
* *Vendas Totais*: Volume financeiro bruto arrecadado.
* *Margem de Lucro*: Percentual de rentabilidade sobre as vendas.
* *Média de Desconto*: Impacto das promoções por subcategoria.
* *Ranking por País*: Identificação dos mercados mais fortes (ex: Estados Unidos liderando com $2,30 Mi).


## 🧠 O que eu aprendi neste projeto:
* Estruturação de um layout focado em *UX (User Experience)* para facilitar a leitura dos dados.
* Aplicação de filtros dinâmicos (Slicers) para análises temporais e geográficas.
* Organização de um repositório profissional para análise de dados.
